# Checklist Pré-Execução Codex - V1.8

Antes de executar qualquer módulo, confirmar:

- [ ] Estou no repositório correto: `CHATBOTCRM`.
- [ ] Li o `README.md` da raiz.
- [ ] Li `docs/backlog/README.md`.
- [ ] Li o Documento Mestre V1.8.
- [ ] Li o módulo atual.
- [ ] Conferi o estado atual do Git com `git status`.
- [ ] Sei quais arquivos pretendo alterar.
- [ ] Não vou alterar `.env` com valores reais.
- [ ] Não vou commitar secrets/tokens/senhas/certificados.
- [ ] Não vou executar migrations destrutivas sem orientação.
- [ ] Não vou avançar para outro módulo.

## Checklist de saída

- [ ] Implementação concluída conforme módulo.
- [ ] Validações executadas.
- [ ] Arquivos alterados listados.
- [ ] Commit pequeno criado ou preparado.
- [ ] Próximo módulo não executado ainda.
