# Checklist Front Codex - V1

Antes de implementar front:

- [ ] Ler README_FRONT.
- [ ] Ler Auditoria UX/UI.
- [ ] Ler Mapa de Navegação.
- [ ] Ler Design System.
- [ ] Ler Estados/Status/Cores.
- [ ] Ler Popups/Modais/Drawers.
- [ ] Conferir imagens em imgs/referencias-front.
- [ ] Confirmar que imagens são guia visual, não regra de negócio.

Durante implementação:

- [ ] Criar componentes reutilizáveis.
- [ ] Evitar duplicação visual.
- [ ] Garantir responsividade mínima.
- [ ] Não hardcodar dados fictícios.
- [ ] Usar dados mockados apenas temporariamente e claramente marcados.
