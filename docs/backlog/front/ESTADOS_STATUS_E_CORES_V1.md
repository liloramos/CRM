# Estados, Status e Cores - V1

## Pedidos

| Status | Cor sugerida | Uso |
| --- | --- | --- |
| Novo | Laranja | Pedido recém-criado. |
| Aguardando pagamento | Amarelo | Cliente precisa pagar/enviar comprovante. |
| Comprovante recebido | Azul | Precisa conferência humana. |
| Pagamento confirmado | Verde | Pedido liberado. |
| Impresso | Cinza/verde | Comanda já impressa. |
| Em preparo | Laranja | Cozinha preparando. |
| Pronto | Verde | Pronto para retirada/entrega. |
| Saiu para entrega | Azul | Em rota. |
| Finalizado | Verde escuro | Pedido encerrado. |
| Cancelado | Vermelho | Pedido cancelado. |
| Manual | Roxo/cinza | Atendente assumiu. |

## Conversas

- Não lida: badge laranja.
- Aguardando cliente: cinza/amarelo.
- IA ativa: verde.
- Atendimento manual: roxo/cinza.
- Atenção necessária: vermelho/laranja.
