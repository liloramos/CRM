# Diagramas do Champs

Esta pasta contém os fluxos de negócio do MVP do Champs.

## Diagramas

1. `01-fluxo-atual.md`  
   Processo manual utilizado antes do sistema.

2. `02-fluxo-prospeccao-mvp.md`  
   Fluxo principal da prospecção no Champs.

3. `03-fluxo-importacao-score-exportacao.md`  
   Fluxo técnico de importação, análise e exportação.

Os diagramas foram inicialmente registrados em Mermaid para facilitar
a revisão e o versionamento.

Após a validação do processo, os fluxos principais poderão ser
transformados em BPMN 2.0 por meio do Camunda Modeler ou Bizagi.
